What is UMNi-NTFY?

UMNi-NTFY is a notification layer for MT5 trading systems. It watches key account and strategy events, then pushes structured updates directly to Telegram so you can stay connected without constantly watching charts.

Notifications include trade activity, online status, heartbeat-style checks, and end-of-session summaries, making it easy to follow what your system is actually doing throughout the day.

Requirements

UMNi-NTFY is designed to run alongside MetaTrader and Telegram. For the current live range, MetaTrader 5 is the primary platform. If you are running a dedicated MetaTrader 4 build, use the matching MT4 version and keep the setup path aligned to that terminal.

  • MetaTrader 5 – required for the current live product release
  • MetaTrader 4 build – use only where a dedicated MT4 version has been supplied
  • Telegram – required for live message delivery
  • Stable internet connection – needed for platform and Telegram communication
  • Demo-first setup – recommended before any live deployment

Core capabilities

  • Real-time trade notifications – instant Telegram updates when trades open or close
  • Session summaries – automated reporting of wins, losses, and net performance
  • Online and heartbeat checks – confirms the system is active and connected
  • Telegram delivery – notifications arrive directly to mobile or desktop Telegram
  • Structured message format – clean notification layout designed for fast reading
  • Lightweight monitoring layer – easy to add into existing MT5 workflows

Notification types

UMNi-NTFY generates structured notifications across the key points of a trading session.

  • System online confirmation
  • Trade opened
  • Trade closed
  • Session summary
  • General status updates

Built for awareness

UMNi-NTFY does not need to replace your trading workflow. It adds a clear, mobile-friendly notification layer so you can follow your MT5 activity with less friction and better visibility.
