Setup Guide

Everything needed to get a UMNi product running cleanly. This guide keeps the launch path simple: install the platform, connect Telegram, attach the product, then verify live delivery before moving into regular use.

Quick start

Step 01

Prepare your trading platform

Install MetaTrader 5 for the current UMNi product range. If you are testing a dedicated MT4 build, use the matching MetaTrader 4 version instead. Log in to your broker or demo account and confirm the platform is connected.

Step 02

Install Telegram

Install Telegram on your phone or desktop, create your account, and open the destination chat or bot you want the product to use for notifications and command delivery.

Step 03

Load the UMNi product

Place the product in the correct platform folder, restart the terminal if needed, and attach it to the intended chart. Match the product build to the platform version you are running: MT5 build for MetaTrader 5, MT4 build for MetaTrader 4.

Step 04

Enter your Telegram details

Open the product inputs, add your Telegram settings, then save the configuration. Once attached, the system should begin sending online, heartbeat, or status output depending on the product tier.

Step 05

Verify live output

Run a test on demo first. Confirm that Telegram messages arrive, the platform stays connected, and the product is reporting the correct account or chart environment before using it in a live setting.

Best practice: Start on a demo account, confirm the full message flow, then move to your live environment only when everything is stable.

Requirements

  • MetaTrader 5 account for the current live UMNi range
  • Telegram account for message delivery and command control
  • Stable internet connection
  • Broker connection active inside the trading terminal
  • Basic understanding of attaching an EA or utility to a chart
